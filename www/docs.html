KFCMAN.CLOUD 이용 및 실시간 공유 가이드

1

모둠원/교실 실시간 공유

1. 파일 카드 우측 아래의 [공유 (링크 아이콘)] 단추를 누릅니다.
2. 생성되는 고유 보안 URL을 복사하여 카카오톡, 디스코드, SMS 등으로 모둠원에게 배포합니다.
3. 링크를 받은 사람은 로그인 없이도 즉시 문서 확인 및 한글(HWP) 원본 다운로드가 가능합니다.

2

공유 및 열람 대상 실시간 추적

1. 복사한 공유 링크를 [메인 URL 단축 서비스]에 등록합니다.
2. 단축 주소 관리 패널에서 [상세 통계]를 확인합니다.
3. 공유받은 상대방의 접속 일시, 접속 IP 주소, Referrer(유입 앱), 접속 기기(모바일/PC 등)를 실시간 로그로 투명하게 추적할 수 있습니다.

복사 완료! 실행 가이드

  1. 에디터 주소가 클립보드에 **복사**되었습니다.
  2. 브라우저 새 탭을 엽니다. (Ctrl + T)
  3. 맨 위 주소창을 클릭 후 **붙여넣기**합니다. (Ctrl + V)
  4. **엔터(Enter)**를 치면 한글 웹에디터가 무결하게 실행됩니다! 🚀
